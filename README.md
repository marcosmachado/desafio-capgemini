# Desafio Capgemini

## Descrição
Este microserviço provê uma API para dentificar se uma sequência de letras é válida.


### Funcionalidades
- Verifica uma sequência válida
- Estatísticas 

## Requisitos
- Java 11
- Gradle 7.5.1
- Mongodb


### Banco de dados
- Baixar uma imagem do mongodb para docker no link https://hub.docker.com/_/mongo
